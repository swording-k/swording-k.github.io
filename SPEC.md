# 杨锟个人作品集网站规格说明书

## 1. 项目概述

- **项目名称**: Yang Kun Portfolio
- **项目类型**: 个人作品集/简历展示网站
- **核心功能**: 展示杨锟的个人背景、实习经历、项目经验与技能
- **目标用户**: 招聘方、合作伙伴、技术社区

## 2. UI/UX 规格

### 2.1 页面结构

采用 **单页垂直滚动 + 锚点导航** 架构，共 5 个主要区块：

1. **Hero 区域** - 首屏全屏展示
2. **关于我** - 个人简介与教育背景
3. **经历** - 实习经历时间线
4. **项目** - 项目作品展示
5. **联系** - 联系方式与底部信息

### 2.2 布局规格

- **最大内容宽度**: 1200px
- **左侧内容区**: 65%
- **右侧装饰区**: 35%（部分区块）
- **垂直内边距**: 120px（桌面）/ 80px（移动端）
- **水平内边距**: 24px

### 2.3 响应式断点

- **桌面**: ≥1024px
- **平板**: 768px - 1023px
- **移动端**: < 768px

### 2.4 色彩方案

采用 **深色高级感** 主题：

```css
--bg-primary: #0a0a0a;        /* 页面主背景 - 近黑色 */
--bg-secondary: #141414;       /* 卡片/区块背景 */
--bg-tertiary: #1f1f1f;       /* 悬浮态背景 */
--text-primary: #f5f5f5;       /* 主文字 - 近白 */
--text-secondary: #a3a3a3;     /* 次要文字 */
--text-muted: #6b6b6b;         /* 辅助文字 */
--accent: #c9a55c;             /* 强调色 - 哑光金 */
--accent-hover: #dfc07a;       /* 强调色悬浮态 */
--border: #2a2a2a;             /* 边框色 */
--gradient-start: #1a1a1a;
--gradient-end: #0a0a0a;
```

### 2.5 字体方案

- **标题字体**: "Playfair Display", serif (优雅衬线体)
- **正文字体**: "DM Sans", sans-serif (现代无衬线)
- **代码/技能**: "JetBrains Mono", monospace

字号规格：
- Hero 标题: 72px / 48px (移动端)
- 区块标题: 42px / 32px (移动端)
- 副标题: 24px / 20px (移动端)
- 正文: 16px
- 小字: 14px

### 2.6 间距系统

- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px
- **3xl**: 64px

### 2.7 视觉效果

- **卡片悬浮**: translateY(-4px) + 边框发光
- **渐变背景**: 从上到下的微妙渐变
- **装饰元素**: 抽象几何图形（圆形、线条）
- **文字动效**: 逐字符淡入
- **滚动动画**: Intersection Observer 触发的淡入上移

### 2.8 组件规格

#### 导航栏
- 固定顶部，背景模糊 (backdrop-filter: blur(12px))
- Logo/姓名 + 导航链接
- 悬浮时背景变亮

#### Hero 区域
- 全屏高度 (100vh)
- 大号姓名标题 + 副标题
- 向下滚动指示器（动画箭头）
- 背景：深色 + 微妙光晕

#### 关于区块
- 左侧：个人简介文字
- 右侧：教育背景时间线
- 照片占位区（需用户提供照片）

#### 经历时间线
- 垂直时间线设计
- 左侧：时间 + 公司
- 右侧：职位 + 详情
- 连接线 + 节点标记

#### 项目卡片
- 网格布局 (2列桌面 / 1列移动)
- 卡片包含：项目名称、角色、描述、关键技术标签
- 悬浮显示更多信息

#### 技能标签
- 圆角胶囊样式
- 背景色区分类别（技术/工具/软技能）

#### 联系区块
- 简洁的联系方式列表
- 社交链接图标

## 4. 技术栈

- **框架**: 纯 HTML + CSS + JavaScript（无框架依赖）
- **字体**: Google Fonts (Playfair Display, DM Sans, JetBrains Mono)
- **图标**: 内联 SVG（无外部图标库依赖）
- **动画**: CSS transitions + Intersection Observer API

## 5. 需要用户手动操作的部分

1. **个人照片**: 需要在 `images/` 目录添加个人照片 (建议 400x400px 方形)
2. **域名配置**: 如需自定义域名，需自行在域名服务商处配置
3. **部署上线**: 可使用 GitHub Pages、Vercel 或 Netlify 部署
